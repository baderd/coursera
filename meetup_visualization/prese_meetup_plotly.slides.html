What is Plotly?¶

  • JavaScript graphing library plotly.js
  • build on top of D3 --> interactivity!
  • plotly.js went open-source in Nov 2015
  • graphs are specified declaratively in JSON format

What else is Plotly?¶

  • API libraries for Python, R, Julia, Scala, ...
  • directly convert ggplot, pandas or matplotlib plots into interactive D3 plots
  • embed interactive Plotly plots into
    • jupyter/ipython notebooks
    • knitr reports
    • Shiny Apps

https://plot.ly/api/

OUTLINE: Grammar of Graphics¶

The Grammar of Graphics is a visualization theory developed by Leland Wilkinson in 1999.

  • influenced the development of graphics and visualization libraries alike
  • focus on 3 of its key principles
    • Separation of data from aesthetics (e.g. in ggplot)
    • Definition of common plot/chart elements
    • Composition of these common elements

Building blocks of visualizations¶

  • visual objects used to encode quantitative values
    • dots
    • lines
    • shapes
  • data types
    • quantitative
    • categorical

Hello world example in Python¶

In [302]:
import plotly.graph_objs as go

trace1 = go.Scatter(
    x = df.categorical.tolist(),
    y = df.quantitative.values,
    mode ='markers')

trace2 = go.Scatter(
    x = df.categorical.tolist(),
    y = df.quantitative.values,
    mode ='lines')

trace3 = go.Bar(
    x = df.categorical.tolist(),
    y = df.quantitative.values)

#define data and layout
data   = [trace1, trace2, trace3]
layout = go.Layout(
    title = "Hello meetup",
    xaxis = dict(title = "categorical"),
    yaxis = dict(title = "quantitative")
)

#define plot
fig = go.Figure(data=data, layout=layout)
p = plot(fig, filename="pp_triple_basics.html", auto_open=False, show_link=False)

%%R
df <- subset(poke_stats, Type %in% c('BUG', 'DRAGON'))
cn_stats = c('HP', 'Attack', 'Defense', 'Speed')
# hierarchical clustering
df <- df[hclust(dist(df[,cn_stats]))$order,]

p <- plot_ly(
    df, y=sapply(df$Name, show_linked_name), x=cn_stats, 
    # give matrix for heatmap
    z=as.matrix(df[,cn_stats]), type='heatmap'
    ) %>% layout(
        # default: sort names alphabetically :-(
        margin=list(l=70), xaxis=list(categoryorder ='trace'), yaxis=list(categoryorder ='trace')
    )
saveWidget(p, "pr_poke_heatmap.html")
